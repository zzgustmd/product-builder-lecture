# Project Blueprint

## Overview

This project is a simple web application that serves as a lottery number generator. It allows users to generate a set of random lottery numbers with the click of a button.

## Features

*   **Lottery Number Generation:** Generates a set of 6 unique random numbers between 1 and 45.
*   **User Interface:** A clean and simple interface to display the generated numbers.
*   **Modern Web Technologies:** Built using HTML, CSS, and JavaScript, following modern web standards.
